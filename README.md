
# 🚧 RoadWatch.

**Track roads. Track accountability.**

---

## 📌 Overview

RoadWatch is a smart road monitoring and complaint system designed to improve **transparency, accountability, and efficiency** in road maintenance.

It allows citizens to **report road issues, track complaint status, and access critical data** like contractor details, budget allocation, and repair history—all in one place.

---

## ❗ Problem

Road infrastructure management suffers from:

* Fragmented and inaccessible data
* Lack of transparency in spending
* No clear accountability for delays
* Inefficient complaint systems

---

## 💡 Solution

RoadWatch bridges the gap by providing a **centralized, user-friendly platform** that integrates:

* Road data
* Complaint tracking
* Budget transparency
* Authority accountability

---

## 🚀 Features

### 🗺️ Map-Based Road Monitoring

* View roads with condition indicators (Good / Moderate / Poor)
* Tap any road to see detailed information

---

### 📍 Road Transparency Dashboard

* Road Type (NH/SH/MDR)
* Contractor details
* Budget sanctioned vs spent
* Last repair date
* Complaint history

---

### 📢 Smart Complaint System

* Report issues with image + location
* Auto-classification of issue type (pothole, crack, etc.)
* Intelligent routing to appropriate authority

---

### 📊 Complaint Tracking (Accountability Layer)

* Real-time status updates
* Timeline view (Submitted → Assigned → In Progress → Completed)
* Delay tracking and authority visibility

---

### 🤖 AI Assistance

* Detects issue severity from images
* Provides insights like expected repair time
* Answers queries about road condition and delays

---

## 🔄 How It Works

1. User opens map and views road conditions
2. Selects a road to see detailed data
3. Reports an issue with image + location
4. System routes complaint to correct authority
5. User tracks progress through timeline

## ⚠️ Note on Data

Due to fragmented availability of structured road data, this project uses a combination of:

* Public government datasets
* Open-source map data
* Simulated data (for demonstration purposes)

The system is designed to integrate real-time official data sources in future.
